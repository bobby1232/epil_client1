# Telegram bot: запись на эпиляцию (MVP)

## Что умеет
- Клиент: запись на услугу (Hold) + телефон + комментарий
- Админ: confirm/reject кнопками
- Авто-истечение Hold (по умолчанию 12 часов)
- Напоминания 24ч/2ч (по умолчанию включены)
- "Мои записи" + отмена по правилу 12 часов

## Запуск локально
1) Создай Postgres (локально или облако)
2) Скопируй `.env.example` -> `.env` и заполни
3) Установи зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4) Старт:
   ```bash
   python -m app.main
   ```

## Прод (Railway)
- Используй WEBHOOK_URL (например https://<your-app>.up.railway.app)
- PORT обычно задаёт Railway автоматически, но можно оставить 8080
- DATABASE_URL должен быть вида: postgresql+asyncpg://...
- SCHEDULE_VISUALIZATION: 1 = стандартный вид слотов, 2 = расписание дня в стиле недельного вида

## Smoke-test
1) /start
2) Записаться -> выбрать услугу -> дату -> время -> телефон -> отправить заявку
3) Админ получает карточку и подтверждает/отклоняет
